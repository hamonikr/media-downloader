# media-downloader

이 프로젝트는 하모니카 OS 에서 유튜브 다운로드나 aria2c, wget 및 사파리 북 같은 도구를 별도의 터미널 옵션을 몰라도 쉽게 사용할 수 있는 프로그램 입니다.

- Auto detect dark theme
- Auto detect locale
- Add Korean translation

upstream : https://github.com/mhogomchungu/media-downloader
